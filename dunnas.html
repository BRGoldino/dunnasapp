  • Cor e tamanho do nome do cliente agora personalizáveis nas configurações do extrato

Produtos padrão do cardápio

Habilita de uma vez todos os produtos do cardápio da Dunnas, sem precisar importar arquivo nenhum. Adiciona os que faltam e atualiza o preço dos que já existem com o mesmo nome — nunca mexe em clientes ou fiados.

Clientes padrão

Cadastra de uma vez a lista de clientes já conhecidos, todos com conta zerada. Só adiciona quem ainda não existe (por nome) — nunca mexe em pedidos, dívidas ou produtos.

Cópia de segurança em texto

Alternativa ao backup .json — um arquivo de texto simples, fácil de ler, com tudo o que está registrado. Não serve pra restaurar no app, é só pra conferência ou guardar uma cópia legível.

Sobre

Desenvolvido por Leonardo Arruda.

Hospedado gratuitamente via GitHub Pages. Os dados ficam salvos só neste aparelho, não em nenhum servidor.

Uso restrito à equipe da Dunnas Alimentação Corporativa.

Opções de desenvolvedor

Envia uma cópia extra de cada backup automático pra um canal do Discord, além de continuar baixando no aparelho normalmente (isso não substitui o backup local, só duplica). O link fica salvo só neste navegador — nunca no código do app.
